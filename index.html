Tentang Saya

Yuk, kenal lebih dekat dengan Saya!

Saya adalah seorang mahasiswa yang bersemangat dalam dunia pengembangan website khusus-nya di bagian Front-end Developer. Dengan keterampilan dan pengetahuan yang saya peroleh selama perjalanan belajar saya, saya menggabungkan kreativitas dan keahlian teknis untuk menciptakan tampilan antarmuka yang menarik dan responsif. Saya selalu bersemangat dalam mengikuti perkembangan terbaru di bidang teknologi web dan senang menciptakan pengalaman pengguna yang luar biasa melalui kode-kode yang saya tulis.

Portfolio

Project Terbaru

Dalam project terbaru ini, saya menghadirkan solusi yang inovatif dan hasil yang luar biasa.

WordPress-Powered Dynamic Website: Commodity Price Data Vicinity IT Del

Website Data Komoditas Harga Kebutuhan Pangan Di Sekitaran IT Del ini dibangun oleh saya dan 2 teman saya. Tujuan website ini dibuat karena sering terjadinya fluktuasi harga di pasar sekitaran IT Del sehingga kami akan menampilkan harga yang tepat, akurat dan disajikan secara Real-time. Agar masyarakat yang berada di sekitaran IT Del mendapat data yang akurat.

VALS Hotel Admin Page Management: Leveraging Laravel for a Dynamic Website

Website VALS Hotel dibangun untuk menawarkan penginapan yang menawarkan banyak layanan kepada penggunanya sehingga saya dan 3 teman saya membangun website ini untuk memudahkan penggunanya untuk memesan VALS Hotel secara online. Pada project ini saya bertugas untuk memenejemen sistem pemesanan kamar yang ada di VALS Hotel.
